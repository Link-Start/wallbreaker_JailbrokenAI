# Fill-in-the-blank post template

## Post 1 (the hook — image attached natively)

```
<Target>: Pwned 🚨
<Target> <version>: Jailbroken📷<mascot-emoji>

<N> of <M> categories. <fires> fires. <ASR-low>-<ASR-high>% ASR.
<one line on what made it interesting — technique CLASS, not the technique itself>

Guardrails down.
```

Mascot-emoji picks (swap for whatever actually fits the target):
- DeepSeek → 🐉   Llama/Meta → 🦙   Grok/xAI → 🛸 or ⚡   Gemini → ♊ or 💎
- Claude/Anthropic → 🌙 or 🅰️   GPT/OpenAI → 🤖 or 🌀   Mistral → 🌬️
- Generic fallback → 🤖

## Reply 1 (self-reply within minutes — the "how", abstracted)

```
How: <2-3 lines on the technique architecture/layers — class of method only,
never operational payload text>.

No jailbreak tax. <one more concrete, surprising detail>.
```

## Reply 2 (self-reply — reply-bait, the highest-leverage single line)

```
<a genuine, answerable question that invites a reply — never rhetorical>
```

Examples: "Which frontier model should we point this at next?" / "What would you
want broken next?" / "Anyone want the defender writeup?"
